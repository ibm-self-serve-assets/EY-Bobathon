Assumptions & Notes

  • Shared in-memory state: InMemoryTaxRepository is a singleton — all test classes within the same factory instance share the same lists. Tests that check absolute counts use ≥ n assertions to remain stable when other tests have added records.
  • No HTTPS in tests: WebApplicationFactory uses plain HTTP internally; UseHttpsRedirection emits a harmless warning but does not affect test outcomes.
  • Calculation math hand-verified: All expected values (tax amounts, rates, risk bands) were derived by tracing TaxCalculationService against seeded fixtures in InMemoryTaxRepository.
  • Risk band edge case: Filing Id=3 (Individual, income $90k) produces an effective rate of 8.33%, which is below the 10% threshold, resulting in Medium (not Low) risk band per the service logic.
  • Program.cs partial class: A public partial class Program {} declaration was appended to Program.cs to expose the entry-point type to WebApplicationFactory<Program>.
